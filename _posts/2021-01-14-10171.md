# 10171번

### 문제

---

이렇게 고양이를 출력

### 입력

---

x

출력

---

```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

단순히 println을 사용하는 방법

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("\\    /\\");
        System.out.println(" )  ( ')");
        System.out.println("(  /  )");
        System.out.println(" \\(__)|");
    }
}

```
